# KeenKeeper

KeenKeeper is a clean and responsive friendship tracking website inspired by a Figma design. It helps users manage meaningful relationships, track communication goals, and monitor interaction history in a simple dashboard-style interface.

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- React Hot Toast

## Key Features

1. Responsive multi-page interface with Home, Friend Details, Timeline, Stats, and 404 page.
2. Quick Check-In system for Call, Text, and Video interactions with timeline logging and toast notifications.
3. Friendship Analytics with a pie chart and interaction timeline filtering for better tracking.
